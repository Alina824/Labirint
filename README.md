# Игра лабиринт с загадками
Игра реализована с помошью библиотеки tkinter языка python. Игра состоит из двух уровней, на которых игроку предстоит пройти лабиринт и ответить на вопросы загадок. По проществию уровня выводится количество звёзд набранных игроком, в зависимости от количества верно разгаданных загадок.
\nПроект старый, реализован в 2019-ом году.
